2. Why do Github users need an SSH key pair? (Users can use either SSH key pair or github account info.) Please write down a brief explanation of SSH key.
github와 함께 사용하고 있을 때, github에게 자신이 사용하고자 알릴 필요가 있는데, 이때 SSH key는 자신의 이름과 비밀번호를 입력할 필요 없이 자신이 사용하고 있음을 알릴 수 있는 방식이다.
SSH key란 SSH(secure shell) 네트워크에 접속가능하게 하는 자격증이라고 할 수 있다.
